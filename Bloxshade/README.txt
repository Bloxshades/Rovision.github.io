Welcome to the BloxShade Utility!

This tool enhances the graphical capabilities of Roblox for a more immersive experience.

Instructions:

Download the contents of this folder.

Run the Setup file.

After the setup finished Wait for the shaders to load into the game engine.

Open Roblox and enjoy your new graphics!

Note: to change the shaders settings press f12 while in game.
